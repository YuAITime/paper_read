2. 问题背景与动机

2.1 人工智能的核心目标

人工智能的一个核心目标是开发能够在不同任务和环境中获取技能的智能体，使用单一、统一的学习范式——直接从周围环境的感官输入中操作，无需手工设计的状态表示或特定领域的校准。

视觉非常适合这一目标：相机价格低廉且可扩展，从像素中学习能够实现从原始感官输入到动作的完全端到端训练。

2.2 JEPA的优势与挑战

Joint Embedding Predictive Architectures (JEPAs) 是一种学习世界模型的流行方法。与尝试建模环境的每个方面不同，JEPA 专注于捕获预测未来状态所需的最相关特征。

具体来说，JEPA 学习将观测编码到紧凑的低维潜在空间，并通过预测未来观测的潜在表示来建模时间动态。

2.3 表征崩溃问题

🚨 核心问题：Representation Collapse

在这种失效模式中，模型将所有输入映射到几乎相同的表示，以平凡地满足时间预测目标，导致无法使用的表示。防止崩溃是训练 JEPA 模型的核心挑战之一。

3. 方法详解：LeWorldModel

3.1 架构设计

🔮

Encoder 编码器

将观测 o 映射到潜在表示 z

z = f(o)

🎯

Predictor 预测器

根据动作预测未来嵌入

ẑₜ₊₁ = g(zₜ, aₜ)

📐

SIGReg 正则化

强制潜在分布为高斯

KL(q(z)||N(0,I))

预测器架构细节

  • • 类型：Transformer，6层，16个注意力头
  • • 参数量：~10M
  • • Dropout：10%
  • • 动作编码：通过 Adaptive Layer Normalization (AdaLN) 在每一层融入
  • • 初始化：AdaLN 参数初始化为零以稳定训练
  • • 掩码：时间因果掩码避免看到未来嵌入

3.2 SIGReg 正则化：核心创新

双损失函数设计

预测损失（Prediction Loss）

Lpred = 𝔼[|| ẑt+1 − zt+1 ||²]

SIGReg 正则化（Single Gaussian Regularization）

Lsigreg = KL( q(z) || 𝒩(0, I) )

强制潜在分布接近标准高斯分布

总损失

L = Lpred + λ · Lsigreg

💡 为什么 SIGReg 有效？

  • 防止 Collapse：高斯分布要求方差为1，防止所有嵌入坍缩到同一点
  • 避免发散：均值为0的约束防止嵌入过度分散
  • 单一超参数：仅需调节 λ，相比 VICReg 的 6+ 超参数大幅降低调参难度
  • 稳定训练：无需 stop-gradient、EMA 等技巧

3.3 训练流程

1

数据收集

从环境中收集观测-动作序列 (oₜ, aₜ, oₜ₊₁)

2

编码

编码器将当前帧和下一帧映射到潜在空间：zₜ = f(oₜ), zₜ₊₁ = f(oₜ₊₁)

3

预测

预测器根据当前潜在表示和动作预测下一时刻：ẑₜ₊₁ = g(zₜ, aₜ)

4

优化

计算总损失 L = L_pred + λ·L_sigreg，端到端优化编码器和预测器

消融实验关键发现

  • • SIGReg 的必要性：移除 SIGReg 导致表征崩溃
  • • λ 的鲁棒性：在 0.01-0.1 范围内性能稳定
  • • 历史帧数：N=8 是性能与效率的最佳平衡点
  • • 嵌入维度：192 维在表示能力和计算效率间取得平衡

7. 深度洞察与未来方向

理论洞察

SIGReg 的成功证明了简单的分布约束可以替代复杂的正则化策略。这与信息瓶颈理论相呼应：通过限制潜在分布的形式，模型被迫学习最相关的信息。

实践意义

LeWM 的极简设计大幅降低了研究门槛。研究者无需深入了解复杂的正则化技巧，也无需进行繁琐的超参数搜索，就能训练出有效的世界模型。

局限与未来方向

  • • 长程规划：当前方法限制在短程规划，层次化世界模型是 promising 方向
  • • 数据依赖：仍需充分的交互数据覆盖，离线数据集收集成本高
  • • 简单环境：在内在维度很低的简单环境中，SIGReg 效果受限
  • • 多模态：扩展到视觉-语言-动作多模态输入
