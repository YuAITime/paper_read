Part I

核心问题与发现

🎯 核心研究问题

"WAMs 是否需要在测试时进行显式的未来想象？
还是其收益主要来自训练时的视频建模？"

关键发现

  • ✓ Fast-WAM 与 imagine-then-execute 性能相当
  • ✓ 移除视频协同训练导致大幅下降
  • ✓ 4倍加速（190ms vs 800ms+）

核心洞察

视频预测的主要价值在于训练时改善世界表示，而非测试时生成未来观测。测试时的视频生成可能是"昂贵的副产品"。

动作专家 (Action Expert)

架构配置

架构类型 DiT (同视频分支)
隐藏维度 d_a 1024
参数量 1B

设计动机

  • 相比视频分支减小维度，降低计算成本
  • 保持相同的 DiT 架构，便于特征融合
  • 专用动作头，优化动作预测精度

训练配置详解

优化器设置

优化器 AdamW
学习率 1×10⁻⁴
权重衰减 0.01
学习率调度 Cosine Annealing
梯度裁剪 1.0
精度 Mixed Precision

训练目标

L = L_action + λ·L_video

联合优化动作预测和视频建模

  • • 使用 Flow Matching 目标
  • • Logit-normal 噪声调度
  • • λ 平衡动作和视频损失
