2. 研究背景

领域定位

本研究属于 Robotics (cs.RO) 领域，聚焦 Vision-Language-Action (VLA) 模型的后训练 (Post-Training) 与持续学习 (Continual Learning)。

研究动机

  • 问题1 - 数据依赖：SFT 需要大量任务特定数据，限制了 VLA 在低数据/少样本场景下的快速适应能力
  • 问题2 - 灾难性遗忘：学习新技能时会退化先前获得的知识，无法支持 VLA 进化为长寿命智能体
  • 核心挑战：提升数据高效适应性往往会加剧遗忘，而保留先验知识又限制了从有限新数据中进行有效学习

解决思路

近期 LLM 研究发现 on-policy 强化学习对遗忘具有更强的鲁棒性。本文探索如何将其应用于 VLA 基础模型，核心挑战在于设计高效、可靠、可扩展的奖励信号。

3. 核心问题定义

研究假设

通过设计无需环境交互的多维过程奖励机制 (MDPR)，结合 chunking-level on-policy 强化学习，可以实现 VLA 模型的持续学习，同时解决数据依赖和灾难性遗忘问题。

验证思路

  1. 在 multi-task 学习场景下验证 LifeLong-RFT 相比 SFT 的性能提升
  2. 在 continual learning 场景下验证 forward transfer 和 backward transfer 指标
  3. 通过消融实验验证 MDPR 三个维度的各自贡献
  4. 在真实机器人环境中验证 sim-to-real 迁移能力

GRPO 目标函数

J_GRPO(θ) = E[(q,o)~D, {a_i}~π_θ_old] [1/G Σ_i min(π_θ/π_θ_old · A_i, clip(π_θ/π_θ_old) · A_i) - β·D_KL(π_θ||π_ref)]

6. 关键创新深入解析：MDPR 三维奖励机制

🎯

L1 · 直观理解

MDPR 像一位"三维裁判"：QACR 检查动作 token 是否猜对，CTAR 检查实际运动轨迹是否准，FCR 检查输出格式是否正确。三者共同确保 VLA 既能准确预测又能正确执行。

🔄

L2 · 机制流程

1. VLA 采样动作块 → 2. QACR 比对 token 一致性 → 3. CTAR 解码后比对轨迹偏差 → 4. FCR 检查格式有效性 → 5. 加权合成 MDPR → 6. GRPO 计算优势并更新策略

🔬

L3 · 技术细节

QACR: Σ_i I(â_i = a_i) / |valid|
CTAR: (1/T)Σ_t[α·exp(-β||p̂_t-p_t||) + I(ĝ_t=g_t)]
FCR: I(valid format)
MDPR: ω·QACR + (1-ω)·CTAR + λ·FCR

QACR - 量化动作一致性奖励

逐位置匹配预测动作 token 与 ground truth，确保离散动作空间中的准确预测。关键：只有格式验证通过的序列才参与计算。

CTAR - 连续轨迹对齐奖励

将 token 解码为连续动作块后，计算与参考轨迹的空间偏差。pose 部分用指数衰减函数，gripper 用二元奖励，加权平均。

FCR - 格式合规奖励

二进制奖励验证输出是否符合 Fast+ tokenizer 的格式要求（动作维度、预测范围等），确保动作可执行。

Part V · 批判性分析

11. 深度批判

深入剖析：为什么有效？

  • On-policy 的优势：使用当前策略采样，数据分布与策略一致，避免离线数据分布偏移；GRPO 的组内相对优势估计减少了方差
  • MDPR 的多维互补：QACR 解决离散空间准确性，CTAR 解决连续执行精度，FCR 解决格式稳定性；三者从语义到物理到结构全面覆盖
  • Env-free 的可扩展性：无需环境交互意味着可以大规模并行，不受 sim-to-real gap 限制
  • Plasticity-Stability 平衡：On-policy RL 天然比 SFT 更抗遗忘，结合 ER (Experience Replay) 实现持续学习

局限性质疑

  • 局限1 - 仅限于离散动作：论文明确承认只针对离散动作模型，连续动作模型性能仍有差距
  • 局限2 - 依赖参考轨迹：CTAR 需要 ground-truth 轨迹计算奖励，在真实场景中可能难以获得
  • 局限3 - 超参数敏感：MDPR 的 ω 和 λ 需要调优，不同任务可能需要不同配置
  • 局限4 - 未与其他 RL 方法充分对比：如 DPO、PPO 等在线 RL 方法的直接对比有限

常见误区澄清

误区澄清
RFT 需要环境交互？LifeLong-RFT 完全不需要环境交互，仅依赖演示数据计算奖励
GRPO = PPO？GRPO 不需要 Critic 网络，通过组内比较估计优势，计算效率更高
QACR 就够了？消融实验证明仅 QACR 不够，CTAR 对物理执行至关重要 (-90.9%)

综合评价

LifeLong-RFT 是 VLA 持续学习领域的重要进展，通过 MDPR 三维奖励机制巧妙解决了无需环境交互的强化微调难题。实验结果扎实，在 multi-task 和 continual learning 上均有显著提升。主要局限在于仅支持离散动作模型，且依赖参考轨迹。未来如能扩展到连续动作空间并开源代码，将更具影响力。
